This folder should contain your custom Perl distribution, if any.
This is also the place to store source code syntax highlighter script - currently 'kate.pl'.

'lib' subfolder should contain all Perl modules you want to ship with your application(s).

'debugger' subfolder should be the place for the Perl debugger output formatter script,
currently 'dbgformatter.pl', as well as for the Win32 Perl debugger patch.
