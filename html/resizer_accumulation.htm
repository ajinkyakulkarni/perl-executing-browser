Image Resizer Script with Accumulation of Results

Select folder Start Script Kill Script

This page uses iframe to display all results from the long-running script. Do not put it inside another iframe in order to see the results!
Select writable folder containing JPG files and start the script - a new folder inside your original folder will be created and smaller versions
of your original files will be written there. They will be only 20% of the original files. ImageMagick 'convert' binary has to be installed in '/usr/bin/'.
The purpose of this (currently Linux-only) example is to demonstrate a simple long-running script inside Perl Executing Browser.
